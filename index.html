누가 어떤 와인을 좋아했는지 바로 읽히는 와인 DB

VVR은 8명의 와인 러버가 남긴 taste 축과 실제 리뷰를 모아 입문자와 애호가가 페르소나별 선호 품종, 지역, 향미 성향, Rich와 Lean 성향, 사진, 가격 메모를 함께 읽을 수 있게 만든 페이지입니다.

이 버전은 Cloudflare Pages에서 바로 배포 가능한 정적 프론트엔드이며, Supabase와 Functions를 연결하면 실제 DB와 외부 데이터 조회로 확장됩니다.

Persona Taste Atlas

Red / White 탭 전환으로 페르소나 취향을 더 읽기 쉽게 정리했습니다.

Wine Review Library

와인별 이미지, 가격 메모, 리뷰어 코멘트를 한 카드에서 확인합니다.

검색

Cloudflare 배포 체크

  1. Cloudflare Pages `Deployments` 탭에서 최신 커밋이 보이는지 확인
  2. `Settings > Environment variables`에 Supabase 키를 추가
  3. `/functions` 엔드포인트가 배포 로그에 포함되는지 확인
